General Info

There are 3 types of artifacts, 'hp', 'dps' and 'all'.

Artifact has a rarity of 1 (green), 2 (yellow), 3 (red). You can upgrade or reset an artifact by regorging, once Forge is unlocked (last building)

Artifacts were meant to only be able to be equipped at the correct 'slot' and by the correct 'type' of units. However the lazy devs never implemnted the check on both sides. So if you want a Mage with a WhetStone, select the unit with WhetStone first and do the switch that way. Same for the 'slots' so you can have a Warrior with 2 Mage staffs using the same 'trick'.

Reforging strategy

A 'Mage' such as WitcherGreen has {{ getNumberOfArtifacts('Mage', 'dps') }} DPS artifacts out of {{ getNumberOfArtifactsTotal('Mage') }} total possible artifacts.

A 'Warrior' such as IronHeart has {{ getNumberOfArtifacts('Warrior', 'dps') }} DPS artifacts out of {{ getNumberOfArtifactsTotal('Warrior') }} total possible artifacts.

A 'Shooter' such as Elf has {{ getNumberOfArtifacts('Shooter', 'dps') }} DPS artifacts out of {{ getNumberOfArtifactsTotal('Shooter') }} total possible artifacts.

This means if you are reforging (green), select a 'Shooter' if you'd like a better chance of a DPS artifact, and a lesser chance of an HP or 3-stat artifact.

Always choose Warrior if you'd like the greatest chance of an all-HP artifact.

Artifact stat calculation

Note: Artifacts are always multiplied by level

rarityMultiplier: 0.20 0.32 0.40

HP artifact calculation:

HP% = rarityMultiplier * level

Example, for a level 3 yellow artifact: .32 * 3= .96%

DPS artifact calculation:

Damage% = Random number between 0-40 Random number between 24-40 40

Spec% = 200 * rarityMulti - damage%

Example simplified:
Rarity 1: Damage is between 0-40, spec is the remaining (total of 40).
Rarity of 2: Damage is 24-40, Spec is remaining (total of 64 since .32 * 200 = 64)
Rarity of 3: Damage is 40, Spec is remaining 40 (total of 80 since .4 * 200 = 80)

Reforging

A random number between 0.0 and 1.0 is generated E.g in browser via `Math.random()`, which is compared to the percentage at cost.

So for 100 yellow/red/greens, the number is compared to 0.10, if the number randomly generated is less than 0.10 the artifact is sucessfuly reforged. For 200, the number is 0.2.
